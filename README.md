# Student Marks Analysis

## Description
A Python-based application that allows users to input student marks and analyze their performance. The program calculates key metrics such as average, highest, and lowest marks using a simple menu-driven approach.

## Features
- Add marks for multiple students
- Calculate average marks
- Find highest and lowest marks
- Input validation for correct data entry
- Handles cases where no data is available

## Technologies Used
- Python

## How to Run
1. Open terminal or command prompt
2. Navigate to the project folder
3. Run the command:
   python marks_analysis.py

## Project Structure
- marks_analysis.py → Main Python program file
